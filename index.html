The Formation

Each agent has a clear mandate.
None with unchecked authority.

Not one model doing everything. Specialized agents with distinct roles, skills, and review criteria.

C
Commander

Coordinates the entire run. Accepts work orders, manages state transitions, and orchestrates handoffs between agents.

P
Planner

Decomposes work into ordered tasks with file paths, function names, and test cases. Precise enough that no questions need to be asked.

B
Builder

Writes code, creates files, and commits changes to a feature branch. Each task gets its own atomic commit.

R
Reviewer

Reviews all commits for code quality, correctness, and adherence to project patterns. Scores and flags issues before QA.

Q
QA

Runs the test suite, measures coverage delta, and verifies all tests pass before the run advances.

S
Security

Scans for vulnerabilities, secret exposure, injection risks, and trust-boundary violations. Blocks the run if critical issues found.

D
Release

Merges the branch, creates the PR, deploys to staging, and runs health checks before marking the run as shipped.

Workflow

A 16-state machine,
not ad-hoc prompts

Every run follows a deterministic path. State transitions are explicit, auditable, and impossible to skip.

INTAKE→ RESEARCHING→ PLANNING→ AWAITING_PLAN_APPROVAL→ EXECUTING→ VERIFYING→ AWAITING_SHIP_APPROVAL→ READY_TO_MERGE→ MERGED→ RELEASE_PREP→ AWAITING_RELEASE_APPROVAL→ SHIPPED

Capabilities

Built for teams that ship

Every feature exists because we needed it building real software.

Config-Driven Workflows

Define agent behavior, approval gates, and skill sets in YAML. Change your process without changing code.

Human Approval Gates

Three mandatory checkpoints: plan, ship, and release approval. Nothing moves forward without your sign-off.

IC-Level Agent Skills

Agents have configurable skill registries — language expertise, framework knowledge, testing patterns. Match agents to your stack.

Project Memory

Postgres + pgvector stores prior runs, decisions, and code context. Agents learn from project history, not just the current prompt.

Guardrails & Safety

Budget limits, token caps, file-scope restrictions, and mandatory security scans. Agents operate within boundaries you define.

Deploy Anywhere

Docker Compose for local dev. Single-command deployment to AWS, GCP, or any cloud. Your infrastructure, your data, your rules.
